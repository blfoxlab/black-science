# 05. Підсумкове завдання модуля

## Що потрібно зробити:

1. **Створити репозиторій** на GitHub під назвою `web-dev-learning`
2. **Склоновати** його на свій комп'ютер
3. **Створити папку** `module-0-setup`
4. **Скопіювати** всі файли цього модуля туди (або створити свої)
5. **Написати README.md** з коротким звітом:
   - Які інструменти встановив
   - Які проблеми були і як вирішив
   - Що нового дізнався
6. Зробити **гарні коміти** з описами
7. Запушити все на GitHub
