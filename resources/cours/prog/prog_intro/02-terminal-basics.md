# 02. Основи роботи з терміналом

## Основні команди

### Навігація по файловій системі

```bash
pwd                    # показати поточну папку
ls                     # список файлів
ls -la                 # детальний список (включаючи приховані)
cd папка               # перейти в папку
cd ..                  # на рівень вище
cd ~                   # перейти в домашню папку
```

### Робота з папками та файлами

```bash
mkdir назва-папки      # створити папку
touch файл.txt         # створити файл
cp файл1.txt файл2.txt # копіювати
mv старий.txt новий.txt # перейменувати або перемістити
rm файл.txt            # видалити файл
rm -rf папка           # видалити папку (обережно!)
```

### Інші корисні команди

```bash
clear                  # очистити термінал (або Ctrl + L)
history                # історія команд
echo "текст"           # вивести текст
cat файл.txt           # показати вміст файлу
```

## Практичні завдання

1. Створи структуру папок для курсу:
```bash
mkdir -p ~/web-dev-course/{module-0,module-1,module-2}
```

2. Попрацюй з файлами:
   - Створи файл `test.txt`
   - Запиши туди текст
   - Скопіюй його
   - Перейменуй копію
   - Видали оригінал

3. Навчись швидко переміщатися між папками.

**Порада:** На Windows рекомендую використовувати **Git Bash** або **PowerShell**.
