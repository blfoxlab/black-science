# 03. Як працює інтернет

## Основні поняття

### 1. Клієнт-Серверна модель
- **Клієнт** — це твій браузер (Chrome, Firefox)
- **Сервер** — комп'ютер, який зберігає сайт і віддає дані

### 2. HTTP та HTTPS
- **HTTP** — протокол обміну даними
- **HTTPS** = HTTP + шифрування (безпека)

**Основні методи HTTP:**
- `GET` — отримати дані
- `POST` — надіслати дані (створення)
- `PUT` — оновити дані повністю
- `PATCH` — часткове оновлення
- `DELETE` — видалення

**Статус-коди:**
- **2xx** — успіх (200 OK, 201 Created)
- **3xx** — перенаправлення
- **4xx** — помилка клієнта (404 Not Found, 401 Unauthorized)
- **5xx** — помилка сервера (500 Internal Server Error)

### 3. DNS (Domain Name System)
Як `google.com` перетворюється на IP-адресу.

### 4. Повний цикл запиту
1. Ти вводиш адресу
2. DNS перетворює домен в IP
3. Браузер відправляє HTTP-запит на сервер
4. Сервер обробляє запит і повертає відповідь
5. Браузер рендерить HTML, CSS, JS

---

## Практика

1. Відкрий DevTools (`F12`) → вкладка **Network**
2. Зайди на google.com або будь-який сайт
3. Проаналізуй:
   - Які файли завантажуються
   - Які запити (GET/POST)
   - Статус-коди
   - Час відповіді
