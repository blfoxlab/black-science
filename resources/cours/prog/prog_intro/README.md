## Чому цей модуль важливий?

- Без правильного налаштування середовища ти будеш постійно витрачати час на технічні проблеми.
- Git — це інструмент, яким ти користуватимешся щодня в реальній роботі.
- Розуміння роботи інтернету допоможе краще розуміти, що відбувається між фронтендом і бекендом.

**Після завершення модуля** ти матимеш повністю готове робоче місце та перший репозиторій на GitHub.
