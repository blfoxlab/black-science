# 06. Практичні завдання та міні-проєкти

## Обов’язкові завдання

1. **Калькулятор** (з підтримкою +, -, *, /)
2. **Гра "Вгадай число"**
3. **Todo List** (додавання, видалення, відмітка виконаних)
4. **Таблиця множення** (генерація через цикли)
5. **Система оцінок** (перетворення балу в оцінку A/B/C/D/F)

## Рекомендовані ресурси для практики
- [Codewars](https://codewars.com) — 8 kyu та 7 kyu
- [javascript.info](https://javascript.info)
- Леока (українською) — YouTube
