# 01. Змінні, типи даних, оператори

## Змінні

```js
// Сучасний спосіб (рекомендовано)
let age = 25;           // можна змінювати
const name = "Олександр"; // не можна змінювати
var oldStyle = "старий"; // не використовувати (hoisting)

// const — обов’язково використовувати за замовчуванням
```

## Типи даних

### Примітивні:
- `string` — "текст"
- `number` — 42, 3.14, Infinity, NaN
- `boolean` — true / false
- `null` — порожнє значення
- `undefined` — значення не присвоєно
- `symbol` (рідко)
- `bigint` (для великих чисел)

### Складні:
- `object` (включаючи масиви, функції, дати)

**Перевірка типу:**
```js
typeof "hello" // "string"
typeof 42      // "number"
```

## Оператори

- Арифметичні: `+ - * / % **`
- Порівняння: `== === != !== > < >= <=`
- Логічні: `&& || !`
- Присвоєння: `= += -= *=`

**Строга рівність (`===`)** — завжди використовуй!
