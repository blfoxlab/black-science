# 06. Практичні проєкти (React)

## Рекомендовані проєкти:

1. **Todo / Task Manager** (з LocalStorage)
2. **Інтернет-магазин** (каталог + кошик)
3. **Погода** (робота з зовнішнім API)
4. **Блог** з постами (CRUD)

**Критерії хорошого проєкту:**
- Використання хуків
- Робота з формами
- Адаптивність (Tailwind)
- Чистий код + компоненти

Після цих проєктів ти будеш готовий до **React Router**, **стан-менеджменту** (Zustand) та переходу на Backend.

Пиши, коли закінчиш — продовжимо курс!
