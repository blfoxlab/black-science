# 04. Основи роботи з Figma

## Що потрібно вміти:

1. Відкривати макети
2. Вимірювати відступи (Hold Alt)
3. Копіювати властивості тексту/кольорів
4. Використовувати Auto Layout
5. Експортувати іконки та зображення
6. Розуміти компоненти та варіанти

## Корисні гарячі клавіші
- `Ctrl + C` / `Ctrl + V` — копіювати стиль
- `I` — Eyedropper (піпетка)
- `R` — Rectangle
- `T` — Text
- `Alt + клік` — виміряти відстань

## Практика

- Візьми безкоштовний макет з [Figma Community](https://www.figma.com/community)
- Спробуй зверстати 1–2 секції точно за макетом

**Рекомендація:** Почни з простих лендінгів (наприклад, Apple-style сторінки).
