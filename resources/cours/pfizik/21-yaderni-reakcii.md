# 21. Ядерні реакції, енергія зв'язку

## Теорія

### Ядерні реакції
Ядерна реакція — процес перетворення атомних ядер при взаємодії з елементарними частинками або іншими ядрами.

Загальний вигляд ядерної реакції:
`_Z1^A1 X + _Z2^A2 Y → _Z3^A3 W + _Z4^A4 V`

### Закони збереження в ядерних реакціях
При будь-якій ядерній реакції зберігаються:
1. Сумарне зарядове число: `Z₁ + Z₂ = Z₃ + Z₄`
2. Сумарне масове число: `A₁ + A₂ = A₃ + A₄`
3. Закон збереження енергії (з урахуванням енергії спокою за формулою Ейнштейна)
4. Закон збереження імпульсу

### Енергетичний вихід ядерної реакції
Різниця енергій спокою (мас) частинок до і після реакції визначає енергію, що виділяється або поглинається:

`Q = (Σm_до - Σm_після)·c²`

Якщо `Q > 0` — реакція екзотермічна (енергія виділяється)
Якщо `Q < 0` — реакція ендотермічна (енергія поглинається, потрібна затрата енергії для її здійснення)

### Реакція поділу ядер
Важкі ядра (наприклад, уран-235) при захопленні нейтрона розділяються на два (рідше більше) ядра середньої маси з виділенням кількох нейтронів і великої кількості енергії:

`_92^235 U + _0^1 n → _56^141 Ba + _36^92 Kr + 3·_0^1 n + Q`

Оскільки при поділі одного ядра вивільняється кілька нейтронів, які можуть викликати поділ інших ядер урану — виникає **ланцюгова реакція**.

**Критична маса** — мінімальна маса ділильного матеріалу, необхідна для підтримання самостійної ланцюгової реакції.

Принцип поділу ядер лежить в основі роботи атомних електростанцій (керована ланцюгова реакція) і ядерної зброї (некерована ланцюгова реакція).

### Реакція термоядерного синтезу
Легкі ядра (наприклад, ізотопи водню) при дуже високих температурах (десятки мільйонів кельвінів) зливаються в більш важкі ядра з виділенням величезної кількості енергії:

`_1^2 H + _1^3 H → _2^4 He + _0^1 n + Q`

Термоядерний синтез — джерело енергії Сонця і зірок; на Землі керований термоядерний синтез поки перебуває на стадії дослідних установок (токамаки).

**Чому потрібна висока температура:** щоб подолати кулонівське відштовхування однойменно заряджених ядер і зблизити їх на відстань дії ядерних сил.

### Порівняння енергетичного виходу
Питома енергія зв'язку менша для дуже легких і дуже важких ядер, і максимальна для ядер середньої маси (наприклад, залізо). Тому енергія виділяється:
- при поділі важких ядер (вони наближаються до максимуму питомої енергії зв'язку)
- при синтезі легких ядер (вони теж наближаються до максимуму)

## Приклади задач

**Задача 1.** У ядерній реакції `_1^2 H + _1^2 H → _2^3 He + _0^1 n` визначити, чи правильно записана реакція за законами збереження.

*Розв'язання:* Зарядові числа: `1+1=2` (зліва), `2+0=2` (справа) ✓
Масові числа: `2+2=4` (зліва), `3+1=4` (справа) ✓
Реакція записана правильно.

**Задача 2.** При ядерній реакції маса продуктів реакції менша за масу початкових частинок на 0,004 а.о.м. Знайти енергію, що виділилась (1 а.о.м. = 1,66·10⁻²⁷ кг).

*Розв'язання:*
`Δm = 0,004·1,66·10⁻²⁷ = 6,64·10⁻³⁰ кг`
`Q = Δm·c² = 6,64·10⁻³⁰·9·10¹⁶ ≈ 5,98·10⁻¹³ Дж`

**Задача 3.** Записати недостаючий продукт реакції: `_92^235 U + _0^1 n → _54^140 Xe + _38^94 Sr + ? ·_0^1 n`

*Розв'язання:*
Зарядові числа: `92+0 = 54+38+0` → `92=92` ✓ (нейтрони заряду не мають)
Масові числа: `235+1 = 140+94+x·1` → `236 = 234+x` → `x=2`
Відповідь: утворюється 2 нейтрони

## Типові помилки та поради
- Завжди перевіряйте баланс зарядових і масових чисел з обох боків рівняння реакції — це основний спосіб знайти невідому частинку
- Не плутайте реакцію поділу (важкі ядра розпадаються на легші) з реакцією синтезу (легкі ядра зливаються у важче)
- Енергія виділяється в обох процесах (і поділ важких, і синтез легких ядер), тому що обидва наближають ядра до максимуму питомої енергії зв'язку
- Термоядерний синтез вимагає надвисоких температур саме для подолання електростатичного відштовхування ядер, а не через якісь інші причини
