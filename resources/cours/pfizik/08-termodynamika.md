# 8. Внутрішня енергія, робота газу, перший закон термодинаміки

## Теорія

### Внутрішня енергія
Внутрішня енергія тіла — сума кінетичної енергії хаотичного руху всіх частинок і потенціальної енергії їх взаємодії.

Для ідеального одноатомного газу внутрішня енергія залежить тільки від температури:
`U = (3/2)·ν·R·T = (3/2)·(m/M)·R·T`

Зміна внутрішньої енергії: `ΔU = (3/2)·ν·R·ΔT`

### Способи зміни внутрішньої енергії
1. Виконання роботи (над газом або газом)
2. Теплообмін (теплопередача)

### Кількість теплоти
`Q = c·m·ΔT`, де `c` — питома теплоємність речовини

### Робота газу
Робота, яку виконує газ при розширенні (сталий тиск):
`A = p·ΔV = p·(V₂ - V₁)`

Якщо газ розширюється — виконує додатну роботу (`A > 0`)
Якщо газ стискається — робота над газом додатна, робота газу від'ємна

Графічно робота газу дорівнює площі під графіком у координатах p-V.

При ізохорному процесі (`V=const`): `A = 0`

### Перший закон термодинаміки
Зміна внутрішньої енергії системи дорівнює сумі кількості теплоти, переданої системі, і роботи, виконаної над системою:

`ΔU = Q + A_над_газом`

або в іншій формі, часто використовуваній на НМТ:

`Q = ΔU + A_газу`

де `A_газу` — робота, яку виконує сам газ.

### Застосування першого закону до ізопроцесів
- **Ізотермічний (T=const):** `ΔU = 0`, тому `Q = A_газу` (вся теплота йде на роботу)
- **Ізохорний (V=const):** `A = 0`, тому `Q = ΔU` (вся теплота йде на зміну внутрішньої енергії)
- **Ізобарний (p=const):** `Q = ΔU + A_газу`
- **Адіабатний процес (Q=0):** `ΔU = -A_газу` (зміна внутрішньої енергії відбувається тільки за рахунок роботи)

## Приклади задач

**Задача 1.** Одноатомний газ кількістю 2 моль нагрівають на 50 К при сталому об'ємі. Знайти зміну внутрішньої енергії.

*Розв'язання:* `ΔU = (3/2)·ν·R·ΔT = 1,5·2·8,31·50 = 1246,5 Дж`

**Задача 2.** Газ при сталому тиску 100 кПа розширюється від об'єму 2 л до 5 л. Знайти роботу газу.

*Розв'язання:* `A = p·ΔV = 100000·0,003 = 300 Дж`

**Задача 3.** Газу передали 500 Дж теплоти, при цьому він виконав роботу 200 Дж. Знайти зміну внутрішньої енергії газу.

*Розв'язання:* `ΔU = Q - A_газу = 500 - 200 = 300 Дж`

**Задача 4.** Газ ізотермічно розширюється, отримавши від нагрівача 400 Дж теплоти. Яку роботу виконав газ?

*Розв'язання:* при T=const, ΔU=0, тому `A_газу = Q = 400 Дж`

## Типові помилки та поради
- Уважно розрізняйте "роботу газу" та "роботу над газом" — вони протилежні за знаком
- При ізохорному процесі робота завжди дорівнює нулю — це часто ключ до розв'язання задачі
- Формула `ΔU = (3/2)νRΔT` працює тільки для одноатомного ідеального газу
- Знак теплоти: якщо газ віддає теплоту, Q від'ємне
