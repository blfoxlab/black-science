# 20. Будова ядра, радіоактивність, закон радіоактивного розпаду

## Теорія

### Будова атомного ядра
Ядро складається з нуклонів: протонів і нейтронів.

**Позначення ядра:** `_Z^A X`, де:
- `Z` — зарядове число (кількість протонів = порядковий номер елемента в таблиці Менделєєва)
- `A` — масове число (загальна кількість нуклонів = протонів + нейтронів)
- Кількість нейтронів: `N = A - Z`

**Ізотопи** — атоми одного хімічного елемента (однакове `Z`) з різною кількістю нейтронів (різне `A`)

### Ядерні сили
Особливий вид взаємодії, що утримує нуклони в ядрі. Характеристики:
- Короткодіючі (діють на відстанях ~10⁻¹⁵ м)
- Значно сильніші за електростатичні (гравітаційні) сили на цих відстанях
- Не залежать від заряду нуклона (діють однаково між протон-протон, протон-нейтрон, нейтрон-нейтрон)

### Дефект маси та енергія зв'язку ядра
Маса ядра завжди менша за суму мас окремих нуклонів, з яких воно складається. Різниця називається дефектом маси:

`Δm = Z·m_p + (A-Z)·m_n - m_ядра`

**Енергія зв'язку ядра** — енергія, яка виділяється при утворенні ядра з окремих нуклонів (або яку потрібно затратити, щоб розділити ядро на окремі нуклони):

`E_зв = Δm·c²`

де `c = 3·10⁸ м/с` — швидкість світла

**Питома енергія зв'язку** (енергія зв'язку на один нуклон) — характеризує стійкість ядра:
`E_пит = E_зв/A`

Найстійкіші ядра — ядра елементів середини таблиці Менделєєва (наприклад, залізо), для них питома енергія зв'язку максимальна.

### Радіоактивність
Явище самовільного перетворення нестійких ядер на інші ядра з випромінюванням частинок.

**Типи радіоактивного розпаду:**

**Альфа-розпад:** ядро випромінює альфа-частинку (ядро гелію `₂⁴He`)
`_Z^A X → _{Z-2}^{A-4} Y + _2^4 He`

**Бета-мінус розпад:** нейтрон перетворюється на протон з випромінюванням електрона (і антинейтрино)
`_Z^A X → _{Z+1}^A Y + _{-1}^0 e`

**Гамма-випромінювання** — випромінювання високоенергетичних фотонів, часто супроводжує альфа- і бета-розпад (ядро переходить зі збудженого стану в основний, при цьому масове і зарядове числа не змінюються)

### Правило зсуву (закони збереження при розпаді)
При будь-якому ядерному перетворенні зберігаються:
- Сумарне зарядове число (сума Z до розпаду = сума Z після розпаду)
- Сумарне масове число (сума A до розпаду = сума A після розпаду)

### Закон радіоактивного розпаду
Кількість ядер, що розпалися, з часом зменшується за експоненційним законом:

`N(t) = N₀·2^(-t/T)`, або `N(t) = N₀·e^(-λt)`

де `N₀` — початкова кількість радіоактивних ядер, `T` — період напіврозпаду (час, за який розпадається половина початкової кількості ядер), `λ` — стала розпаду

Активність джерела (кількість розпадів за одиницю часу) зменшується за тим самим законом.

## Приклади задач

**Задача 1.** Ядро урану `₉₂²³⁸U` зазнає альфа-розпаду. Визначити ядро, що утворюється.

*Розв'язання:* `Z' = 92-2 = 90`, `A' = 238-4 = 234` → утворюється торій `₉₀²³⁴Th`

**Задача 2.** Період напіврозпаду ізотопу становить 8 днів. Скільки відсотків початкової кількості ядер залишиться через 24 дні?

*Розв'язання:* `24/8 = 3` періоди напіврозпаду; `N/N₀ = (1/2)³ = 1/8 = 12,5%`

**Задача 3.** Дефект маси ядра дорівнює 0,03 а.о.м. (1 а.о.м. = 1,66·10⁻²⁷ кг). Знайти енергію зв'язку ядра.

*Розв'язання:* 
`Δm = 0,03·1,66·10⁻²⁷ = 4,98·10⁻²⁹ кг`
`E_зв = Δm·c² = 4,98·10⁻²⁹·9·10¹⁶ ≈ 4,48·10⁻¹² Дж`

## Типові помилки та поради
- При написанні ядерних реакцій завжди перевіряйте баланс зарядових і масових чисел з обох боків рівняння
- Не плутайте альфа- і бета-розпад: при альфа-розпаді зарядове число зменшується на 2, масове — на 4; при бета-мінус розпаді зарядове число збільшується на 1, масове число не змінюється
- Кількість періодів напіврозпаду — це `n = t/T`, а частка ядер, що залишилися: `(1/2)ⁿ`
- Енергія зв'язку завжди позитивна величина — це енергія, яку потрібно затратити для розділення ядра на нуклони
