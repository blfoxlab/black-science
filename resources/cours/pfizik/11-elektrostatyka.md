# 11. Електростатика: закон Кулона, напруженість поля, потенціал, конденсатори

## Теорія

### Електричний заряд
Заряд квантований: `q = n·e`, де `e = 1,6·10⁻¹⁹ Кл` — елементарний заряд

Закон збереження заряду: у замкненій системі сумарний заряд не змінюється

### Закон Кулона
Сила взаємодії між двома точковими зарядами:
`F = k·|q₁·q₂|/r²`

де `k = 9·10⁹ Н·м²/Кл²` (у вакуумі), `r` — відстань між зарядами

Однойменні заряди відштовхуються, різнойменні — притягуються.

### Напруженість електричного поля
`E = F/q` (сила, що діє на одиничний позитивний заряд)

Напруженість поля точкового заряду:
`E = k·|Q|/r²`

**Принцип суперпозиції:** напруженість результуючого поля дорівнює векторній сумі напруженостей від кожного заряду окремо.

Напруженість однорідного поля між пластинами конденсатора:
`E = U/d`, де `d` — відстань між пластинами

### Потенціал і потенціальна енергія
Потенціал точки поля: `φ = W_п/q`

Потенціал поля точкового заряду: `φ = k·Q/r`

**Напруга (різниця потенціалів):**
`U = φ₁ - φ₂ = A/q`

Робота сил електричного поля при переміщенні заряду:
`A = q·U`

Зв'язок напруженості та напруги в однорідному полі:
`U = E·d`

### Провідники і діелектрики в електричному полі
- У провіднику вільні заряди перерозподіляються так, що поле всередині провідника дорівнює нулю (електростатична індукція)
- Діелектрики поляризуються під дією зовнішнього поля, що послаблює поле всередині них

### Електроємність. Конденсатори
Електроємність — здатність провідника накопичувати заряд:
`C = q/U`

**Плоский конденсатор:**
`C = ε·ε₀·S/d`

де `ε` — діелектрична проникність середовища, `ε₀ = 8,85·10⁻¹² Ф/м`, `S` — площа пластин, `d` — відстань між ними

### З'єднання конденсаторів
**Послідовне з'єднання:**
`1/C = 1/C₁ + 1/C₂ + ...`
`q₁ = q₂ = q` (заряд однаковий на всіх конденсаторах)

**Паралельне з'єднання:**
`C = C₁ + C₂ + ...`
`U₁ = U₂ = U` (напруга однакова)

### Енергія зарядженого конденсатора
`W = C·U²/2 = q²/(2C) = q·U/2`

## Приклади задач

**Задача 1.** Два точкові заряди по 2·10⁻⁶ Кл кожен розташовані на відстані 0,3 м. Знайти силу взаємодії.

*Розв'язання:* `F = k·q₁q₂/r² = 9·10⁹·2·10⁻⁶·2·10⁻⁶/0,09 = 0,4 Н`

**Задача 2.** Знайти напруженість поля точкового заряду 5·10⁻⁸ Кл на відстані 10 см.

*Розв'язання:* `E = kQ/r² = 9·10⁹·5·10⁻⁸/0,01 = 45000 В/м`

**Задача 3.** Плоский конденсатор ємністю 2 мкФ заряджений до напруги 100 В. Знайти заряд і енергію конденсатора.

*Розв'язання:* `q = C·U = 2·10⁻⁶·100 = 2·10⁻⁴ Кл`; `W = CU²/2 = 2·10⁻⁶·10000/2 = 0,01 Дж`

**Задача 4.** Два конденсатори по 4 мкФ з'єднані послідовно. Знайти загальну ємність.

*Розв'язання:* `1/C = 1/4 + 1/4 = 1/2` → `C = 2 мкФ`

## Типові помилки та поради
- При послідовному з'єднанні конденсаторів загальна ємність менша за найменшу з ємностей — перевіряйте себе за цим правилом
- Не плутайте формули для послідовного з'єднання конденсаторів і резисторів — вони протилежні!
- Напруженість поля — векторна величина, застосовуйте принцип суперпозиції з урахуванням напрямків
- У задачах з декількома зарядами уважно визначайте знаки зарядів і напрямки сил/напруженостей
