# 9. Теплові явища: теплоємність, фазові переходи, вологість повітря

## Теорія

### Теплоємність
**Питома теплоємність** `c` — кількість теплоти, потрібна для нагрівання 1 кг речовини на 1 К.

Кількість теплоти при нагріванні/охолодженні:
`Q = c·m·(t₂ - t₁)`

### Рівняння теплового балансу
Для системи тіл без втрат теплоти в навколишнє середовище:
`Q_відданої = Q_отриманої`

### Фазові переходи

**Плавлення і кристалізація**
Питома теплота плавлення `λ` — кількість теплоти, потрібна для перетворення 1 кг речовини з твердого стану в рідкий при температурі плавлення:
`Q = λ·m`

При кристалізації виділяється така сама кількість теплоти.

**Пароутворення і конденсація**
Питома теплота пароутворення `L` (або `r`) — кількість теплоти, потрібна для перетворення 1 кг рідини в пару при температурі кипіння:
`Q = L·m`

При конденсації виділяється така сама кількість теплоти.

**Згоряння палива**
Питома теплота згоряння `q`:
`Q = q·m`

### Графік нагрівання речовини
На графіку залежності температури від часу (або кількості теплоти) горизонтальні ділянки відповідають фазовим переходам (температура не змінюється, поки вся речовина не перейде в інший стан).

### Вологість повітря

**Абсолютна вологість** — маса водяної пари в 1 м³ повітря (густина пари), позначається `ρ` (кг/м³ або г/м³)

**Відносна вологість:**
`φ = (ρ/ρ_нас)·100% = (p/p_нас)·100%`

де `ρ_нас` — густина насиченої пари при даній температурі, `p_нас` — тиск насиченої пари

**Точка роси** — температура, при якій пара, що міститься в повітрі, стає насиченою (починається конденсація)

## Приклади задач

**Задача 1.** Скільки теплоти потрібно для нагрівання 2 кг води від 20°C до 80°C? (c_води = 4200 Дж/(кг·К))

*Розв'язання:* `Q = c·m·Δt = 4200·2·60 = 504000 Дж = 504 кДж`

**Задача 2.** Скільки теплоти потрібно для плавлення 0,5 кг льоду при температурі плавлення? (λ_льоду = 3,3·10⁵ Дж/кг)

*Розв'язання:* `Q = λ·m = 3,3·10⁵·0,5 = 165000 Дж = 165 кДж`

**Задача 3.** У калориметр з водою масою 200 г при температурі 20°C кинули шматок металу масою 100 г, нагрітий до 100°C. Кінцева температура становила 25°C. Знайти питому теплоємність металу (знехтувати теплоємністю калориметра, c_води=4200 Дж/(кг·К)).

*Розв'язання:* Теплота, отримана водою = теплоті, відданій металом:
`c_води·m_води·(25-20) = c_мет·m_мет·(100-25)`
`4200·0,2·5 = c_мет·0,1·75`
`4200 = 7,5·c_мет`
`c_мет = 560 Дж/(кг·К)`

**Задача 4.** Відносна вологість повітря 60% при тиску насиченої пари 2,3 кПа. Знайти парціальний тиск пари в повітрі.

*Розв'язання:* `φ = p/p_нас` → `p = φ·p_нас = 0,6·2,3 = 1,38 кПа`

## Типові помилки та поради
- У задачах на теплообмін завжди перевіряйте, чи потрібно враховувати фазові переходи (наприклад, лід тане, потім вода нагрівається)
- Не забувайте про теплоємність посудини (калориметра), якщо вона задана в умові
- Відносна вологість завжди виражається у відсотках і не може перевищувати 100%
- При підвищенні температури тиск насиченої пари зростає нелінійно (використовуйте таблиці, якщо надано в завданні)
