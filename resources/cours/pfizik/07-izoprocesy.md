# 7. Ізопроцеси (ізотермічний, ізобарний, ізохорний), графіки

## Теорія

Ізопроцеси — процеси зміни стану газу сталої маси, при яких один із параметрів (T, p або V) залишається незмінним.

### Ізотермічний процес (T = const)
Закон Бойля — Маріотта: `p·V = const`, тобто `p₁·V₁ = p₂·V₂`

Графік у координатах (p,V) — гіпербола (ізотерма)
Графік у координатах (p,T) або (V,T) — вертикальна пряма

### Ізобарний процес (p = const)
Закон Гей-Люссака: `V/T = const`, тобто `V₁/T₁ = V₂/T₂`

Графік у координатах (V,T) — пряма лінія, що проходить через початок координат (у шкалі Кельвіна)
Графік у координатах (p,V) — вертикальна пряма
Графік у координатах (p,T) — горизонтальна пряма

### Ізохорний процес (V = const)
Закон Шарля: `p/T = const`, тобто `p₁/T₁ = p₂/T₂`

Графік у координатах (p,T) — пряма лінія, що проходить через початок координат
Графік у координатах (p,V) — горизонтальна пряма
Графік у координатах (V,T) — вертикальна пряма

### Об'єднаний газовий закон
Для газу сталої маси при переході з одного стану в інший:
`(p₁·V₁)/T₁ = (p₂·V₂)/T₂`

### Читання графіків на НМТ
Важливо вміти визначати тип процесу за графіком і за потрібними координатами перебудовувати графік з одних координат в інші (p-V, V-T, p-T).

**Алгоритм:**
1. Визначте, які осі використані на графіку
2. Визначте, який параметр сталий (горизонтальна/вертикальна лінія або тип кривої)
3. Застосуйте відповідний закон

## Приклади задач

**Задача 1.** Газ при тиску 100 кПа займає об'єм 4 л. Тиск ізотермічно збільшили до 200 кПа. Знайти новий об'єм газу.

*Розв'язання:* `p₁V₁ = p₂V₂` → `V₂ = p₁V₁/p₂ = 100·4/200 = 2 л`

**Задача 2.** Газ при температурі 300 К займає об'єм 2 л при сталому тиску. Газ нагрівають до 400 К. Знайти новий об'єм.

*Розв'язання:* `V₁/T₁ = V₂/T₂` → `V₂ = V₁·T₂/T₁ = 2·400/300 ≈ 2,67 л`

**Задача 3.** Тиск газу при сталому об'ємі і температурі 250 К дорівнює 150 кПа. Знайти тиск при температурі 500 К.

*Розв'язання:* `p₁/T₁ = p₂/T₂` → `p₂ = p₁·T₂/T₁ = 150·500/250 = 300 кПа`

**Задача 4.** Газ переходить зі стану (p₁=200 кПа, V₁=3 л, T₁=300 К) у стан (p₂=?, V₂=6 л, T₂=600 К). Знайти p₂.

*Розв'язання:* `(p₁V₁)/T₁ = (p₂V₂)/T₂` → `p₂ = (p₁V₁T₂)/(V₂T₁) = (200·3·600)/(6·300) = 200 кПа`

## Типові помилки та поради
- Завжди переводьте температуру в кельвіни перед розрахунками
- На НМТ часто дають графіки в незвичних координатах (наприклад, p-T замість V-T) — тренуйтеся розпізнавати тип процесу за будь-якою парою координат
- Пам'ятайте: пряма лінія через початок координат означає пряму пропорційність, а не будь-яка пряма лінія
- Для складних процесів (наприклад, комбінація двох ізопроцесів) розбивайте графік на етапи і аналізуйте кожен окремо
