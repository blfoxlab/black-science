# Координати та вектори в просторі

## 9.1 Координати в просторі

### 9.1.1 Прямокутна система координат в просторі

Має три взаємно перпендикулярні осі: `Ox`, `Oy`, `Oz`.

### 9.1.2 Координати точки

Точка записується як `A(x; y; z)`.

### 9.1.3 Відстань між точками

```text
AB = √((x2-x1)^2 + (y2-y1)^2 + (z2-z1)^2)
```

### 9.1.4 Середина відрізка

```text
M((x1+x2)/2; (y1+y2)/2; (z1+z2)/2)
```

## 9.2 Вектори в просторі

### 9.2.1 Поняття вектора в просторі

Вектор у просторі має три координати: `(x; y; z)`.

### 9.2.2 Операції з векторами

Додавання, віднімання і множення на число виконують покоординатно.

### 9.2.3 Скалярний добуток

```text
a·b = a1b1 + a2b2 + a3b3
```

### 9.2.4 Кут між векторами

```text
cos φ = (a·b)/(|a||b|)
```

### 9.2.5 Колінеарні вектори

Вектори колінеарні, якщо їх координати пропорційні.

### 9.2.6 Компланарні вектори

Три вектори компланарні, якщо лежать в одній або паралельних площинах.

### 9.2.7 Розкладання вектора за базисом

Будь-який вектор простору можна подати через три некомпланарні базисні вектори.

## 9.3 Рівняння площини

### 9.3.1 Через точку та нормальний вектор

Якщо нормальний вектор `n(A;B;C)`, а точка `M0(x0;y0;z0)`, то:

```text
A(x-x0) + B(y-y0) + C(z-z0) = 0
```

### 9.3.2 Загальне рівняння площини

```text
Ax + By + Cz + D = 0
```

### 9.3.3 Через три точки

Площину можна задати трьома точками, що не лежать на одній прямій.

## 9.4 Рівняння прямої в просторі

### 9.4.1 Параметричне рівняння

```text
x = x0 + at
y = y0 + bt
z = z0 + ct
```

### 9.4.2 Канонічне рівняння

```text
(x-x0)/a = (y-y0)/b = (z-z0)/c
```

### 9.4.3 Рівняння через дві точки

Напрямний вектор прямої дорівнює різниці координат двох точок.
