# Декартова система координат

## 5.1 Координати на площині

### 5.1.1 Прямокутна система координат

Складається з двох перпендикулярних осей `Ox` і `Oy`.

### 5.1.2 Координати точки

Точка записується як `A(x; y)`, де `x` - абсциса, `y` - ордината.

### 5.1.3 Відстань між точками

```text
AB = √((x2 - x1)^2 + (y2 - y1)^2)
```

### 5.1.4 Середина відрізка

```text
M((x1+x2)/2; (y1+y2)/2)
```

## 5.2 Рівняння фігур

### 5.2.1 Рівняння прямої

```text
y = kx + b
Ax + By + C = 0
```

### 5.2.2 Рівняння кола

```text
(x - a)^2 + (y - b)^2 = R^2
```

### 5.2.3 Рівняння еліпса, параболи, гіперболи

Для НМТ найважливіша парабола `y = ax^2 + bx + c`. Еліпс і гіпербола зазвичай трапляються як поглиблена тема.

## 5.3 Трансформації на площині

### 5.3.1 Паралельне перенесення

Усі точки фігури зміщуються на один і той самий вектор.

### 5.3.2 Поворот

Фігура повертається навколо точки на заданий кут.

### 5.3.3 Симетрія

Осьова симетрія відбиває фігуру відносно прямої. Центральна - відносно точки.

### 5.3.4 Гомотетія

Гомотетія змінює розміри фігури в `k` разів, зберігаючи форму.

## 5.4 Вектори на площині

### 5.4.1 Поняття вектора

Вектор має довжину й напрям.

### 5.4.2 Операції з векторами

```text
(a1; a2) + (b1; b2) = (a1+b1; a2+b2)
k(a1; a2) = (ka1; ka2)
```

### 5.4.3 Скалярний добуток

```text
a·b = a1b1 + a2b2 = |a||b|cos φ
```

### 5.4.4 Кут між векторами

```text
cos φ = (a·b)/(|a||b|)
```

### 5.4.5 Умови паралельності та перпендикулярності

Перпендикулярність: `a·b = 0`. Паралельність: координати пропорційні.

### 5.4.6 Розкладання вектора за базисом

Будь-який вектор площини можна подати як лінійну комбінацію двох неколінеарних векторів.
