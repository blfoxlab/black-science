# Рівняння та системи рівнянь

## 5.1 Лінійні рівняння

Лінійне рівняння має вигляд `ax + b = 0`.

### 5.1.1 Розв'язування лінійних рівнянь

Якщо `a != 0`, то:

```text
x = -b/a
```

Якщо `a = 0`, треба окремо перевірити `b`: при `b = 0` розв'язків безліч, при `b != 0` їх немає.

### 5.1.2 Рівняння з параметрами

Параметр - це літера, яку тимчасово вважають сталою. У таких задачах треба розглянути особливі випадки, наприклад коли коефіцієнт при `x` дорівнює нулю.

### 5.1.3 Геометрична інтерпретація

Лінійне рівняння з двома змінними `ax + by = c` задає пряму. Розв'язок системи двох таких рівнянь - точка перетину прямих.

## 5.2 Квадратні рівняння

### 5.2.1 Повні квадратні рівняння

Повне квадратне рівняння:

```text
ax^2 + bx + c = 0, a != 0
```

### 5.2.2 Неповні квадратні рівняння

Якщо немає одного з коефіцієнтів:

- `ax^2 + bx = 0` -> `x(ax + b) = 0`;
- `ax^2 + c = 0` -> `x^2 = -c/a`;
- `ax^2 = 0` -> `x = 0`.

### 5.2.3 Дискримінант

```text
D = b^2 - 4ac
```

Якщо `D > 0` - два корені, `D = 0` - один корінь, `D < 0` - дійсних коренів немає.

### 5.2.4 Формула коренів

```text
x = (-b ± √D)/(2a)
```

### 5.2.5 Теорема Вієта

Для `ax^2 + bx + c = 0`:

```text
x1 + x2 = -b/a
x1*x2 = c/a
```

Для зведеного рівняння `x^2 + px + q = 0`: `x1 + x2 = -p`, `x1*x2 = q`.

### 5.2.6 Розкладання квадратного тричлена

Якщо корені `x1`, `x2`, то:

```text
ax^2 + bx + c = a(x - x1)(x - x2)
```

## 5.3 Рівняння вищих степенів

### 5.3.1 Біквадратні рівняння

Мають вигляд `ax^4 + bx^2 + c = 0`. Заміна: `t = x^2`, `t >= 0`.


### 5.3.4 Метод заміни змінної

Якщо у рівнянні повторюється одна й та сама складна частина, познач її новою змінною. Після розв'язання обов'язково повернися до початкової змінної.

### 5.3.5 Теорема про раціональні корені

Для многочлена з цілими коефіцієнтами можливі раціональні корені часто шукають серед дільників вільного члена.

## 5.4 Ірраціональні рівняння

### 5.4.1 Розв'язування рівнянь з коренями

Алгоритм: ОДЗ, ізоляція кореня, піднесення до степеня, розв'язання, перевірка.

### 5.4.2 Перевірка розв'язків

Перевірка обов'язкова, бо піднесення до парного степеня може додати зайві корені.

### 5.4.3 Побічні корені

Побічний корінь задовольняє перетворене рівняння, але не початкове. Його треба відкинути.

## 5.5 Дробово-раціональні рівняння

### 5.5.1 Розв'язування дробово-раціональних рівнянь

Зводимо дроби до спільного знаменника або множимо обидві частини на спільний знаменник.

### 5.5.2 ОДЗ дробово-раціональних рівнянь

Усі знаменники не дорівнюють нулю. Це записують до розв'язання.

### 5.5.3 Методи розв'язування

Основні методи: спільний знаменник, розкладання на множники, заміна змінної, перевірка ОДЗ.

## 5.6 Показникові рівняння

### 5.6.1 Основні методи розв'язування

Приведення до однієї основи, винесення спільного множника, заміна змінної, логарифмування.

### 5.6.2 Рівняння виду `a^f(x) = a^g(x)`

Якщо `a > 0`, `a != 1`, то:

```text
f(x) = g(x)
```

### 5.6.3 Рівняння виду `a^f(x) = b`

Якщо `b` можна подати як степінь основи `a`, прирівнюємо показники. Якщо ні - логарифмуємо.

### 5.6.4 Рівняння з різними основами

Шукай спільну основу: `4 = 2^2`, `8 = 2^3`, `9 = 3^2`, `27 = 3^3`.

## 5.7 Логарифмічні рівняння

### 5.7.1 Основні методи розв'язування

ОДЗ, властивості логарифмів, перехід до показникової форми, перевірка.

### 5.7.2 `log_a(f(x)) = log_a(g(x))`

За однакової основи:

```text
f(x) = g(x)
```

але тільки після запису ОДЗ: `f(x) > 0`, `g(x) > 0`.

### 5.7.3 `log_a(f(x)) = b`

Переходимо до показникової форми:

```text
f(x) = a^b
```

### 5.7.4 ОДЗ логарифмічних рівнянь

Аргументи логарифмів мають бути додатними, основи - додатними й не рівними `1`.

## 5.8 Тригонометричні рівняння

### 5.8.1 `sin x = a`

Розв'язки існують при `|a| <= 1`:

```text
x = (-1)^n arcsin a + πn
```

### 5.8.2 `cos x = a`

```text
x = ± arccos a + 2πn
```

### 5.8.3 `tg x = a`

```text
x = arctg a + πn
```

### 5.8.4 `ctg x = a`

```text
x = arcctg a + πn
```

### 5.8.5 Однорідні тригонометричні рівняння

Однорідні рівняння містять однаковий сумарний степінь `sin x` і `cos x`. Часто ділять на `cos^n x` або `sin^n x`, не забуваючи перевірити випадки, коли дільник дорівнює нулю.

### 5.8.6 Рівняння, що розв'язуються розкладанням на множники

Якщо рівняння має вигляд добутку:

```text
f(x)g(x) = 0
```

то `f(x) = 0` або `g(x) = 0`.

## 5.9 Системи рівнянь

### 5.9.1 Системи лінійних рівнянь

Розв'язок системи - пара чисел, що задовольняє всі рівняння одночасно.

### 5.9.2 Методи розв'язування

Підстановка зручна, коли одну змінну легко виразити. Додавання зручне, коли коефіцієнти при одній змінній протилежні або легко такими стають.

### 5.9.3 Метод Крамера

Для системи двох рівнянь:

```text
D = a1b2 - a2b1
x = Dx/D
y = Dy/D
```

де `D != 0`.

### 5.9.4 Системи нелінійних рівнянь

Часто розв'язуються підстановкою, додаванням, відніманням або заміною змінної.

### 5.9.5 Системи змішаного типу

Наприклад, пряма й парабола. Зручно підставити лінійний вираз у нелінійне рівняння.
