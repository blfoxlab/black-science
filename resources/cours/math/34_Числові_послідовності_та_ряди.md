# Числові послідовності та ряди

## 2.1 Числові ряди

### 2.1.1 Збіжність та розбіжність

Ряд - сума нескінченної послідовності. Якщо часткові суми мають скінченну границю, ряд збігається.

### 2.1.2 Необхідна умова збіжності

Якщо ряд збігається, його загальний член прямує до нуля.

### 2.1.3 Ознаки збіжності

Ознаки порівняння, Даламбера й Коші допомагають визначити збіжність без точного знаходження суми.

### 2.1.4 Знакозмінні ряди

Містять додатні й від'ємні члени. Для них важлива ознака Лейбніца.

### 2.1.5 Функціональні ряди

Члени ряду є функціями від `x`.

### 2.1.6 Степеневі ряди

Мають вигляд суми степенів `(x-a)^n` з коефіцієнтами.

## 2.2 Розвинення функцій в ряди Тейлора та Маклорена

### 2.2.1 Формула Тейлора

Функцію наближують многочленом через похідні в точці.

### 2.2.2 Ряд Маклорена для `e^x`

```text
e^x = 1 + x + x^2/2! + x^3/3! + ...
```

### 2.2.3 Ряд Маклорена для `sin x`

```text
sin x = x - x^3/3! + x^5/5! - ...
```

### 2.2.4 Ряд Маклорена для `cos x`

```text
cos x = 1 - x^2/2! + x^4/4! - ...
```

### 2.2.5 Ряд Маклорена для `ln(1+x)`

```text
ln(1+x) = x - x^2/2 + x^3/3 - ...
```

при відповідних значеннях `x`.
