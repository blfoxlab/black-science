# Випадкові величини

## 3.1 Дискретні випадкові величини

### 3.1.1 Визначення

Дискретна випадкова величина набуває окремих значень.

### 3.1.2 Закон розподілу

Показує, з якими ймовірностями величина набуває кожного значення.

### 3.1.3 Функція розподілу

```text
F(x) = P(X <= x)
```

### 3.1.4 Таблиця розподілу

У таблиці записують значення `x_i` та ймовірності `p_i`, причому `Σp_i = 1`.

## 3.2 Неперервні випадкові величини

### 3.2.1 Визначення

Неперервна випадкова величина може набувати будь-якого значення з проміжку.

### 3.2.2 Функція розподілу

Так само `F(x)=P(X<=x)`.

### 3.2.3 Щільність розподілу

Ймовірність на проміжку знаходять інтегралом від щільності.

### 3.2.4 Властивості щільності

Щільність невід'ємна, а площа під усім графіком дорівнює `1`.

## 3.3 Числові характеристики

### 3.3.1 Математичне сподівання

Для дискретної величини:

```text
M(X) = Σx_i p_i
```

### 3.3.2 Властивості

`M(aX+b)=aM(X)+b`.

### 3.3.3 Дисперсія

```text
D(X)=M((X-MX)^2)
```

### 3.3.4 Властивості дисперсії

`D(aX+b)=a^2D(X)`.

### 3.3.5 Середньоквадратичне відхилення

```text
σ = √D(X)
```

### 3.3.6 Коваріація та кореляція

Описують зв'язок між двома випадковими величинами.

### 3.3.7 Коефіцієнт кореляції

Належить проміжку `[-1;1]`.

## 3.4 Розподіли ймовірностей

### 3.4.1 Рівномірний

Усі значення або рівні проміжки мають однакову ймовірність.

### 3.4.2 Нормальний

Дзвоноподібний розподіл, важливий у статистиці.

### 3.4.3 Експоненціальний

Описує час очікування до події.

### 3.4.4 Біноміальний

Кількість успіхів у схемі Бернуллі.

### 3.4.5 Пуассона

Кількість рідкісних подій за фіксований проміжок.

### 3.4.6 Стьюдента

Використовують для малих вибірок.

### 3.4.7 Хі-квадрат

Застосовується у статистичних критеріях.

### 3.4.8 Фішера

Використовується для порівняння дисперсій.
