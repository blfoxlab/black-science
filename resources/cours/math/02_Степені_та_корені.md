# Степені та корені

## 2.1 Степеневі вирази

Степінь показує повторене множення. Якщо основа `a`, а показник `n`, то `a^n` читається як "`a` у степені `n`".

### 2.1.1 Натуральний показник степеня

Для натурального `n`:

```text
a^n = a * a * ... * a
```

Наприклад, `2^4 = 2*2*2*2 = 16`.

### 2.1.2 Нульовий показник

Для `a != 0`:

```text
a^0 = 1
```

Важливо: `0^0` у шкільному курсі зазвичай не розглядають як визначений вираз.

### 2.1.3 Цілий від'ємний показник

Для `a != 0`:

```text
a^(-n) = 1/a^n
```

Наприклад, `2^(-3) = 1/8`.

### 2.1.4 Дробовий показник

Дробовий показник пов'язаний з коренем:

```text
a^(m/n) = n√(a^m)
```

Для парного `n` основа має бути невід'ємною, якщо працюємо в дійсних числах.

### 2.1.5 Ірраціональний показник

Степінь з ірраціональним показником означають через наближення раціональними показниками. Для НМТ достатньо знати, що `a^x` при `a > 0`, `a != 1` задає показникову функцію.

## 2.2 Властивості степенів

Властивості степенів працюють за умов, коли вирази мають зміст. Найчастіше треба слідкувати, щоб основа не була нулем у діленні й від'ємному степені.

### 2.2.1 Добуток степенів з однією основою

```text
a^m * a^n = a^(m+n)
```

### 2.2.2 Частка степенів з однією основою

```text
a^m / a^n = a^(m-n), a != 0
```

### 2.2.3 Степінь добутку

```text
(ab)^n = a^n b^n
```

### 2.2.4 Степінь частки

```text
(a/b)^n = a^n/b^n, b != 0
```

### 2.2.5 Степінь степеня

```text
(a^m)^n = a^(mn)
```

Типова пастка: `a^m + a^n` не дорівнює `a^(m+n)`. Правила степенів працюють для множення й ділення, а не для додавання.

## 2.3 Коренева функція

Корінь `n`-го степеня з числа `a` - це число, `n`-й степінь якого дорівнює `a`.

### 2.3.1 Квадратний корінь

`√a` - невід'ємне число, квадрат якого дорівнює `a`. Область визначення: `a >= 0`.

### 2.3.2 Кубічний корінь

`∛a` існує для всіх дійсних `a`. Наприклад, `∛27 = 3`, `∛(-27) = -3`.

### 2.3.3 Корінь n-го степеня

Якщо `n` парне, то `n√a` існує при `a >= 0`. Якщо `n` непарне, то існує для будь-якого дійсного `a`.

### 2.3.4 Властивості коренів

```text
n√(ab) = n√a * n√b
n√(a/b) = n√a / n√b
n√(a^n) = |a|, якщо n парне
n√(a^n) = a, якщо n непарне
```

### 2.3.5 Розв'язування рівнянь з коренями

Алгоритм:

1. Записати ОДЗ.
2. Ізолювати корінь.
3. Піднести обидві частини до потрібного степеня.
4. Розв'язати отримане рівняння.
5. Перевірити корені в початковому рівнянні.

Приклад: `√(x + 1) = 3`. ОДЗ: `x + 1 >= 0`. Підносимо до квадрата: `x + 1 = 9`, отже `x = 8`.
