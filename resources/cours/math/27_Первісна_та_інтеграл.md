# Первісна та інтеграл

## 4.1 Поняття первісної

### 4.1.1 Визначення первісної

`F(x)` - первісна для `f(x)`, якщо `F'(x) = f(x)`.

### 4.1.2 Основна теорема про первісні

Якщо `F` - первісна, то всі первісні мають вигляд `F(x) + C`.

### 4.1.3 Неозначений інтеграл

```text
∫f(x)dx = F(x) + C
```

## 4.2 Властивості неозначеного інтегралу

### 4.2.1 Інтеграл суми

```text
∫(f + g)dx = ∫f dx + ∫g dx
```

### 4.2.2 Інтеграл добутку на константу

```text
∫k f(x)dx = k∫f(x)dx
```

### 4.2.3 Інтеграл різниці

```text
∫(f - g)dx = ∫f dx - ∫g dx
```

## 4.3 Таблиця інтегралів

```text
∫x^n dx = x^(n+1)/(n+1) + C, n != -1
∫sin x dx = -cos x + C
∫cos x dx = sin x + C
∫e^x dx = e^x + C
∫a^x dx = a^x/ln a + C
∫dx/x = ln|x| + C
∫dx/(1+x^2) = arctg x + C
∫dx/√(1-x^2) = arcsin x + C
```

## 4.4 Методи інтегрування

### 4.4.1 Метод розкладання

Розкласти складний вираз на суму простих.

### 4.4.2 Метод підставлення

Замінити внутрішній вираз новою змінною.

### 4.4.3 Інтегрування частинами

```text
∫u dv = uv - ∫v du
```

### 4.4.4 Раціональні функції

Часто розкладають на прості дроби або спрощують діленням многочленів.

### 4.4.5 Ірраціональні функції

Використовують підстановки й властивості коренів.

### 4.4.6 Тригонометричні функції

Застосовують тотожності й заміну.

## 4.5 Означений інтеграл

### 4.5.1 Визначення

Означений інтеграл має межі `a` і `b`.

### 4.5.2 Геометричний смисл

Це орієнтована площа під графіком.

### 4.5.3 Фундаментальна теорема числення

```text
∫_a^b f(x)dx = F(b) - F(a)
```

### 4.5.4 Властивості

Можна розбивати проміжок, виносити константу, інтегрувати суму почленно.

### 4.5.5 Обчислення

Знайти первісну, підставити верхню межу, відняти значення в нижній межі.

## 4.6 Застосування означеного інтегралу

### 4.6.1 Площа плоскої фігури

```text
S = ∫_a^b (f(x) - g(x))dx
```

де `f >= g`.

### 4.6.2 Довжина дуги кривої

Поглиблена формула: `L = ∫√(1+(y')^2)dx`.

### 4.6.3 Об'єм тіла обертання

Метод дисків: `V = π∫(f(x))^2 dx`.

### 4.6.4 Площа поверхні обертання

Поглиблена тема через інтеграли.

### 4.6.5 Фізичні застосування

Інтеграл може задавати шлях за швидкістю, роботу сили, накопичену величину.
