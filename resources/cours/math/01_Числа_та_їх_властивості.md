# Числа та їх властивості

## 1.1 Натуральні числа

Натуральні числа - це числа для лічби: `1, 2, 3, ...`. У задачах НМТ вони найчастіше з'являються в темах подільності, остач, дільників, кратних, простих чисел.

### 1.1.1 Операції з натуральними числами

Основні дії: додавання, віднімання, множення, ділення. Додавання й множення завжди дають натуральне число, а віднімання та ділення - не завжди. Наприклад, `3 - 5` не є натуральним, а `7 : 2` не є натуральним.

### 1.1.2 Ділення і дільники

Число `b` є дільником числа `a`, якщо `a` ділиться на `b` без остачі. Запис: `a : b` або `b | a`. Наприклад, дільники числа `12`: `1, 2, 3, 4, 6, 12`.

Кратне числа `b` - це число вигляду `bk`, де `k` натуральне. Наприклад, кратні `5`: `5, 10, 15, 20, ...`.

### 1.1.3 Прості і складені числа

Просте число має рівно два натуральні дільники: `1` і саме себе. Приклади: `2, 3, 5, 7, 11`. Число `2` - єдине парне просте число. Складене число має більше двох дільників: `4, 6, 8, 9, 10`.

### 1.1.4 Розкладання на прості множники

Будь-яке складене натуральне число можна розкласти на добуток простих множників. Наприклад:

```text
60 = 2^2 * 3 * 5
```

Це використовують для знаходження `НСД`, `НСК`, скорочення дробів і перевірки подільності.

### 1.1.5 НОД і НОК

`НСД` - найбільший спільний дільник. `НСК` - найменше спільне кратне. Через розклад:

- для `НСД` беремо спільні прості множники з меншими степенями;
- для `НСК` беремо всі прості множники з більшими степенями.

Приклад: `36 = 2^2*3^2`, `48 = 2^4*3`. Тоді `НСД = 2^2*3 = 12`, `НСК = 2^4*3^2 = 144`.

## 1.2 Цілі числа

Цілі числа: `..., -3, -2, -1, 0, 1, 2, 3, ...`. Вони потрібні для роботи зі знаками, координатами, температурами, боргами, змінами величин.

### 1.2.1 Операції з цілими числами

При додаванні чисел однакового знака додаємо модулі й лишаємо знак. При додаванні різних знаків від більшого модуля віднімаємо менший і ставимо знак числа з більшим модулем.

Множення й ділення:

- `+ * + = +`
- `- * - = +`
- `+ * - = -`
- `- * + = -`

### 1.2.2 Модуль числа

Модуль `|a|` - це відстань від числа `a` до нуля. Тому модуль ніколи не буває від'ємним:

```text
|a| = a, якщо a >= 0
|a| = -a, якщо a < 0
```

### 1.2.3 Властивості цілих чисел

Цілі числа замкнені щодо додавання, віднімання і множення, але не завжди щодо ділення. Наприклад, `8 : 4 = 2` - ціле, а `5 : 2 = 2.5` - не ціле.

## 1.3 Раціональні числа

Раціональні числа - це числа, які можна подати як дріб `m/n`, де `m` - ціле, `n` - натуральне. Сюди входять цілі числа, звичайні дроби, скінченні та періодичні десяткові дроби.

### 1.3.1 Дроби звичайні

Звичайний дріб має чисельник і знаменник: `a/b`, де `b != 0`. Правильний дріб має чисельник менший за знаменник. Неправильний - чисельник більший або рівний знаменнику.

### 1.3.2 Дроби десяткові

Десятковий дріб - запис числа через кому. Наприклад, `0,25 = 25/100 = 1/4`. Скінченний десятковий дріб завжди раціональний.

### 1.3.3 Переведення дробів

Щоб звичайний дріб перевести в десятковий, треба поділити чисельник на знаменник. Щоб десятковий перевести у звичайний, записуємо число без коми в чисельнику, а в знаменнику ставимо `10`, `100`, `1000` залежно від кількості цифр після коми.

### 1.3.4 Операції з раціональними числами

Для додавання дробів потрібен спільний знаменник:

```text
a/b + c/d = (ad + bc)/bd
a/b - c/d = (ad - bc)/bd
a/b * c/d = ac/bd
a/b : c/d = a/b * d/c
```

### 1.3.5 Порівняння раціональних чисел

Дроби можна порівнювати зведенням до спільного знаменника або перехресним множенням. Якщо `b > 0`, `d > 0`, то `a/b > c/d` тоді й тільки тоді, коли `ad > bc`.

## 1.4 Ірраціональні числа

Ірраціональні числа не можна подати у вигляді дробу цілих чисел. Приклади: `√2`, `√3`, `π`. Їх десятковий запис нескінченний і неперіодичний.

### 1.4.1 Квадратні корені

Квадратний корінь `√a` - невід'ємне число, квадрат якого дорівнює `a`. Для дійсних чисел `√a` існує лише при `a >= 0`.

### 1.4.2 Кубічні корені

Кубічний корінь `∛a` існує для будь-якого дійсного `a`, бо куб може бути і додатним, і від'ємним. Наприклад, `∛(-8) = -2`.

### 1.4.3 Властивості коренів

```text
√(ab) = √a * √b
√(a/b) = √a / √b
√(a^2) = |a|
```

Для парного кореня обов'язково перевіряємо невід'ємність підкореневого виразу.

### 1.4.4 Звільнення від ірраціональності

Якщо в знаменнику є корінь, його часто прибирають множенням на спряжений вираз. Наприклад:

```text
1/√3 = √3/3
1/(√5 - 2) = (√5 + 2)/(5 - 4) = √5 + 2
```

## 1.5 Дійсні числа

Дійсні числа - це всі раціональні та ірраціональні числа. Їх можна зображати на числовій прямій.

### 1.5.1 Основні властивості

Для дійсних чисел виконуються переставна, сполучна і розподільна властивості:

```text
a + b = b + a
ab = ba
(a + b) + c = a + (b + c)
a(b + c) = ab + ac
```

### 1.5.2 Порядок операцій

Спочатку виконують дії в дужках, потім степені й корені, потім множення та ділення, потім додавання й віднімання.

### 1.5.3 Точність і наближення

Округлення залежить від наступної цифри: якщо вона `0-4`, лишаємо; якщо `5-9`, збільшуємо попередню цифру на `1`. У прикладних задачах треба зважати на одиниці вимірювання й умову округлення.
