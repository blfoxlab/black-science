# Многогранники

## 7.1 Загальні властивості многогранників

### 7.1.1 Грані, ребра, вершини

Многогранник обмежений многокутниками. Многокутники - грані, їх сторони - ребра, кінці ребер - вершини.

### 7.1.2 Формула Ейлера

Для опуклого многогранника:

```text
В - Р + Г = 2
```

де `В` - вершини, `Р` - ребра, `Г` - грані.

### 7.1.3 Опуклість

Многогранник опуклий, якщо відрізок між будь-якими двома його точками повністю лежить у ньому.

## 7.2 Призма

### 7.2.1 Визначення та класифікація

Призма має дві рівні паралельні основи, а бічні грані - паралелограми.

### 7.2.2 Пряма та похила призма

У прямій призмі бічні ребра перпендикулярні до основ. У похилій - ні.

### 7.2.3 Правильна призма

Пряма призма, основою якої є правильний многокутник.

### 7.2.4 Площа бічної поверхні

Для прямої призми:

```text
S_біч = P_осн*h
```

### 7.2.5 Площа повної поверхні

```text
S_повн = S_біч + 2S_осн
```

### 7.2.6 Об'єм призми

```text
V = S_осн*h
```

### 7.2.7 Діагональ призми

Для прямокутного паралелепіпеда:

```text
d = √(a^2 + b^2 + c^2)
```

## 7.3 Паралелепіпед

### 7.3.1 Прямий паралелепіпед

Бічні ребра перпендикулярні до основи.

### 7.3.2 Похилий паралелепіпед

Бічні ребра не перпендикулярні до основи.

### 7.3.3 Прямокутний паралелепіпед

Усі грані - прямокутники.

### 7.3.4 Куб

Усі ребра рівні, усі грані - квадрати.

### 7.3.5 Об'єм паралелепіпеда

```text
V = abc
```

для прямокутного паралелепіпеда.

### 7.3.6 Діагоналі

Діагональ прямокутного паралелепіпеда: `d = √(a^2+b^2+c^2)`.

## 7.4 Піраміда

### 7.4.1 Визначення та класифікація

Піраміда має одну основу, а бічні грані - трикутники зі спільною вершиною.

### 7.4.2 Правильна піраміда

Основа - правильний многокутник, вершина проєктується в центр основи.

### 7.4.3 Площа бічної поверхні

Для правильної піраміди:

```text
S_біч = P_осн*l/2
```

### 7.4.4 Площа повної поверхні

```text
S_повн = S_біч + S_осн
```

### 7.4.5 Об'єм піраміди

```text
V = S_осн*h/3
```

### 7.4.6 Апофема

Апофема - висота бічної грані правильної піраміди.

### 7.4.7 Висота

Висота - перпендикуляр із вершини до площини основи.

## 7.5 Зрізана піраміда

### 7.5.1 Визначення та властивості

Утворюється перетином піраміди площиною, паралельною основі.

### 7.5.2 Площа бічної поверхні

Для правильної зрізаної піраміди:

```text
S_біч = (P1 + P2)l/2
```

### 7.5.3 Площа повної поверхні

```text
S_повн = S_біч + S1 + S2
```

### 7.5.4 Об'єм зрізаної піраміди

```text
V = h(S1 + √(S1S2) + S2)/3
```

## 7.6 Правильні многогранники

### 7.6.1 Тетраедр

Має 4 рівносторонні трикутні грані.

### 7.6.2 Куб

Має 6 квадратних граней.

### 7.6.3 Октаедр

Має 8 рівносторонніх трикутних граней.

### 7.6.4 Додекаедр

Має 12 правильних п'ятикутних граней.

### 7.6.5 Ікосаедр

Має 20 рівносторонніх трикутних граней.
