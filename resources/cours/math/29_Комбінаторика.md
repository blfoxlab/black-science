# Комбінаторика

## 1.1 Основні правила комбінаторики

### 1.1.1 Правило суми

Якщо дію можна виконати одним із кількох несумісних способів, кількості додаються.

### 1.1.2 Правило добутку

Якщо дія складається з послідовних етапів, кількості варіантів перемножуються.

### 1.1.3 Принцип включення-виключення

Для двох множин:

```text
|A ∪ B| = |A| + |B| - |A ∩ B|
```

## 1.2 Перестановки

### 1.2.1 Визначення

Перестановка - упорядкування всіх `n` елементів.

### 1.2.2 Формула

```text
P_n = n!
```

### 1.2.3 Факторіал

```text
n! = 1*2*...*n
0! = 1
```

### 1.2.4 Перестановки з повтореннями

```text
P = n!/(n1!n2!...nk!)
```

## 1.3 Розміщення

### 1.3.1 Визначення

Розміщення - упорядкований вибір `k` елементів з `n`.

### 1.3.2 Без повторень

```text
A_n^k = n!/(n-k)!
```

### 1.3.3 З повтореннями

```text
n^k
```

## 1.4 Комбінації

### 1.4.1 Визначення

Комбінація - вибір `k` елементів з `n` без урахування порядку.

### 1.4.2 Без повторень

```text
C_n^k = n!/(k!(n-k)!)
```

### 1.4.3 З повтореннями

```text
C_(n+k-1)^k
```

### 1.4.4 Властивості

```text
C_n^k = C_n^(n-k)
```

### 1.4.5 Трикутник Паскаля

Кожне число дорівнює сумі двох над ним.

### 1.4.6 Біном Ньютона

```text
(a+b)^n = Σ C_n^k a^(n-k)b^k
```

## 1.5 Впорядковані та невпорядковані вибірки

### 1.5.1 З повтореннями та без

Перед вибором формули з'ясуй: порядок важливий чи ні, повторення дозволені чи ні.

### 1.5.2 Задачі на вибір

Порядок важливий - розміщення. Порядок не важливий - комбінації. Усі елементи - перестановки.
