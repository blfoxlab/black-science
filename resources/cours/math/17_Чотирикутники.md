# Чотирикутники

## 3.1 Загальні властивості чотирикутника

### 3.1.1 Сума кутів

Сума внутрішніх кутів будь-якого чотирикутника дорівнює `360°`.

### 3.1.2 Периметр та площа

Периметр - сума сторін. Площу знаходять за формулами для конкретного виду чотирикутника або розбиттям на трикутники.

### 3.1.3 Діагоналі

Діагональ з'єднує несусідні вершини. Властивості діагоналей часто визначають тип чотирикутника.

## 3.2 Паралелограм

### 3.2.1 Визначення та властивості

Паралелограм - чотирикутник, у якого протилежні сторони попарно паралельні. Протилежні сторони й кути рівні, діагоналі діляться навпіл.

### 3.2.2 Ознаки паралелограма

Якщо протилежні сторони попарно рівні або діагоналі діляться навпіл, чотирикутник є паралелограмом.

### 3.2.3 Площа паралелограма

```text
S = ah = ab sin α
```

### 3.2.4 Висота

Висота - перпендикуляр між паралельними сторонами.

## 3.3 Прямокутник

### 3.3.1 Визначення та властивості

Прямокутник - паралелограм з усіма прямими кутами.

### 3.3.2 Діагоналі

Діагоналі прямокутника рівні й діляться навпіл.

### 3.3.3 Площа

```text
S = ab
```

### 3.3.4 Периметр

```text
P = 2(a + b)
```

## 3.4 Ромб

### 3.4.1 Визначення та властивості

Ромб - паралелограм з усіма рівними сторонами.

### 3.4.2 Діагоналі та їх властивості

Діагоналі ромба перпендикулярні, діляться навпіл і ділять кути ромба навпіл.

### 3.4.3 Площа

```text
S = ah = d1*d2/2
```

### 3.4.4 Периметр

```text
P = 4a
```

## 3.5 Квадрат

### 3.5.1 Визначення та властивості

Квадрат - одночасно прямокутник і ромб.

### 3.5.2 Діагоналі

```text
d = a√2
```

Діагоналі рівні, перпендикулярні, діляться навпіл і ділять кути навпіл.

### 3.5.3 Площа

```text
S = a^2 = d^2/2
```

### 3.5.4 Периметр

```text
P = 4a
```

## 3.6 Трапеція

### 3.6.1 Визначення та властивості

Трапеція - чотирикутник із однією парою паралельних сторін.

### 3.6.2 Рівнобічна трапеція

Бічні сторони рівні, кути при кожній основі рівні, діагоналі рівні.

### 3.6.3 Прямокутна трапеція

Має два прямі кути.

### 3.6.4 Середня лінія трапеції

```text
m = (a + b)/2
```

### 3.6.5 Площа трапеції

```text
S = (a + b)h/2
```

### 3.6.6 Периметр

Сума всіх сторін.

## 3.7 Правильні многокутники

### 3.7.1 Визначення та властивості

Правильний многокутник має всі сторони й усі кути рівні.

### 3.7.2 Центральний кут

```text
360°/n
```

### 3.7.3 Внутрішній кут правильного многокутника

```text
180°(n - 2)/n
```

### 3.7.4 Сума внутрішніх кутів

```text
180°(n - 2)
```

### 3.7.5 Площа правильного многокутника

```text
S = Pr/2
```

де `r` - радіус вписаного кола.

### 3.7.6 Периметр

```text
P = na
```
