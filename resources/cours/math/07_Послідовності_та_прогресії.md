# Послідовності та прогресії

## 7.1 Числові послідовності

Числова послідовність - це набір чисел, розташованих у певному порядку: `a1, a2, a3, ...`. Кожен елемент має номер.

### 7.1.1 Визначення послідовності

Послідовність можна розглядати як функцію натурального аргументу: номеру `n` відповідає число `a_n`.

### 7.1.2 Способи задання послідовностей

Основні способи: формула `a_n`, рекурентне правило, таблиця або словесний опис. Рекурентно означає, що наступний член залежить від попередніх.

### 7.1.3 Монотонні послідовності

Послідовність зростає, якщо `a_(n+1) > a_n`, і спадає, якщо `a_(n+1) < a_n`.

### 7.1.4 Обмежені послідовності

Послідовність обмежена зверху, якщо всі її члени не перевищують деякого числа. Обмежена знизу - якщо всі члени не менші за деяке число.

### 7.1.5 Границя послідовності

Число `A` є границею послідовності, якщо її члени при великих `n` наближаються до `A`.

## 7.2 Арифметична прогресія

Арифметична прогресія - послідовність, у якій кожен наступний член отримують додаванням сталої різниці `d`.

### 7.2.1 Визначення та властивості

```text
a_(n+1) = a_n + d
```

Будь-який член, крім першого й останнього, є середнім арифметичним сусідніх:

```text
a_n = (a_(n-1) + a_(n+1))/2
```

### 7.2.2 Формула n-го члена

```text
a_n = a_1 + (n - 1)d
```

### 7.2.3 Сума n перших членів

```text
S_n = (a_1 + a_n)n/2
S_n = (2a_1 + (n - 1)d)n/2
```

### 7.2.4 Середнє арифметичне

Середнє арифметичне чисел `a` і `b`:

```text
(a + b)/2
```

### 7.2.5 Задачі на арифметичну прогресію

Типовий алгоритм: визначити `a_1`, `d`, `n`, вибрати формулу для `a_n` або `S_n`, підставити дані.

## 7.3 Геометрична прогресія

Геометрична прогресія - послідовність, у якій кожен наступний член отримують множенням на сталий знаменник `q`.

### 7.3.1 Визначення та властивості

```text
b_(n+1) = b_n * q
```

Для додатних сусідніх членів: `b_n^2 = b_(n-1)b_(n+1)`.

### 7.3.2 Формула n-го члена

```text
b_n = b_1 q^(n - 1)
```

### 7.3.3 Сума n перших членів

```text
S_n = b_1(q^n - 1)/(q - 1), q != 1
```

### 7.3.4 Нескінченна геометрична прогресія

Нескінченна геометрична прогресія має суму лише тоді, коли `|q| < 1`.

### 7.3.5 Сума нескінченної геометричної прогресії

```text
S = b_1/(1 - q), |q| < 1
```

### 7.3.6 Задачі на геометричну прогресію

Шукай фразу "кожного разу збільшується/зменшується у стільки-то разів" - це сигнал геометричної прогресії.

## 7.4 Границі послідовностей та рядів

### 7.4.1 Границя послідовності

Якщо `a_n` наближається до числа `A`, пишуть `lim a_n = A`.

### 7.4.2 Теореми про границі

Границя суми дорівнює сумі границь, добутку - добутку границь, частки - частці границь, якщо знаменник не прямує до нуля.

### 7.4.3 Число e як границя

```text
e = lim(1 + 1/n)^n
```

Це число є основою натурального логарифма.
