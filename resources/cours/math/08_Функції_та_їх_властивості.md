# Функції та їх властивості

## 8.1 Основні поняття функції

Функція - правило, яке кожному допустимому `x` ставить у відповідність рівно одне значення `y`.

### 8.1.1 Визначення функції

Запис `y = f(x)` означає, що значення `y` залежить від `x` за правилом `f`.

### 8.1.2 Область визначення

Область визначення - усі `x`, для яких вираз має зміст. Обмеження: знаменник не нуль, парний корінь має невід'ємний підкореневий вираз, логарифм має додатний аргумент.

### 8.1.3 Область значень

Область значень - усі можливі `y`, які може приймати функція.

### 8.1.4 Графік функції

Графік - множина точок `(x; f(x))` на координатній площині.

### 8.1.5 Способи задання функції

Функцію задають формулою, таблицею, графіком або словесним описом.

## 8.2 Властивості функцій

### 8.2.1 Парність та непарність

Парна: `f(-x) = f(x)`, графік симетричний відносно `Oy`. Непарна: `f(-x) = -f(x)`, графік симетричний відносно початку координат.

### 8.2.2 Монотонність

Функція зростає, якщо більшому `x` відповідає більше `y`. Спадає, якщо більшому `x` відповідає менше `y`.

### 8.2.3 Обмеженість

Функція обмежена зверху, якщо `f(x) <= M`, і знизу, якщо `f(x) >= m`.

### 8.2.4 Найбільше та найменше значення

На відрізку неперервна функція досягає найбільшого й найменшого значень. Їх шукають у критичних точках і на кінцях відрізка.

### 8.2.5 Періодичність

Функція періодична, якщо існує `T != 0`, що `f(x + T) = f(x)`.

### 8.2.6 Опуклість та увігнутість

Опуклість і вгнутість описують вигин графіка. У старшому курсі це досліджують через другу похідну.

## 8.3 Лінійна функція

### 8.3.1 Формула та графік

```text
y = kx + b
```

Графік - пряма.

### 8.3.2 Кутовий коефіцієнт

`k` показує нахил прямої. Якщо `k > 0`, функція зростає; якщо `k < 0`, спадає.

### 8.3.3 Умови паралельності та перпендикулярності

Паралельні прямі мають однакові `k`. Перпендикулярні прямі мають `k1*k2 = -1`.

### 8.3.4 Пряма, що проходить через дві точки

Кутовий коефіцієнт:

```text
k = (y2 - y1)/(x2 - x1)
```

## 8.4 Квадратична функція

### 8.4.1 Формула та графік

```text
y = ax^2 + bx + c
```

Графік - парабола.

### 8.4.2 Вершина параболи

```text
x0 = -b/(2a)
y0 = f(x0)
```

### 8.4.3 Вісь симетрії

```text
x = -b/(2a)
```

### 8.4.4 Нулі функції

Нулі - розв'язки рівняння `ax^2 + bx + c = 0`.

### 8.4.5 Монотонність

Якщо `a > 0`, функція спадає до вершини й зростає після неї. Якщо `a < 0` - навпаки.

### 8.4.6 Розташування параболи

Залежить від знака `a`, дискримінанта й вершини.

## 8.5 Степенева функція

### 8.5.1 Властивості при n натуральному

`y = x^n`. Якщо `n` парне, функція парна; якщо непарне - непарна.

### 8.5.2 Властивості при n цілому від'ємному

`y = x^(-n) = 1/x^n`, тому `x != 0`.

### 8.5.3 Властивості при n дробовому

Пов'язана з коренями. Для парних коренів область визначення часто `x >= 0`.

### 8.5.4 Графіки степеневих функцій

Треба знати базові графіки: `x`, `x^2`, `x^3`, `1/x`, `√x`.

## 8.6 Показникова функція

### 8.6.1 Визначення та властивості

```text
y = a^x, a > 0, a != 1
```

Область визначення: усі дійсні числа. Область значень: `y > 0`.

### 8.6.2 Графіки для `a > 1` та `0 < a < 1`

При `a > 1` функція зростає. При `0 < a < 1` - спадає.

### 8.6.3 Монотонність

Монотонність важлива для показникових нерівностей.

### 8.6.4 Практичні застосування

Показникові функції описують складні відсотки, ріст населення, радіоактивний розпад.

## 8.7 Логарифмічна функція

### 8.7.1 Визначення та властивості

```text
y = log_a x, a > 0, a != 1
```

Область визначення: `x > 0`.

### 8.7.2 Графіки для `a > 1` та `0 < a < 1`

При `a > 1` функція зростає. При `0 < a < 1` - спадає.

### 8.7.3 Монотонність

Монотонність визначає, чи змінюється знак нерівності при переході від логарифмів до аргументів.

### 8.7.4 Зв'язок з показниковою функцією

Логарифмічна функція є оберненою до показникової.

## 8.8 Обернена функція

### 8.8.1 Визначення оберненої функції

Обернена функція міняє місцями аргумент і значення: якщо `y = f(x)`, то `x = f^(-1)(y)`.

### 8.8.2 Існування оберненої функції

Обернена функція існує, якщо кожного значення `y` функція набуває не більше одного разу. Достатньо, щоб функція була строго монотонною.

### 8.8.3 Графік оберненої функції

Графіки `f` і `f^(-1)` симетричні відносно прямої `y = x`.

### 8.8.4 Властивості оберненої функції

Область визначення оберненої функції - область значень початкової, і навпаки.

## 8.9 Композиція функцій

### 8.9.1 Складна функція

Композиція `f(g(x))` означає, що спочатку обчислюють `g(x)`, потім застосовують `f`.

### 8.9.2 Знаходження області визначення складної функції

Треба врахувати умови для внутрішньої функції `g(x)` і для зовнішньої функції `f`, куди підставлено `g(x)`.
