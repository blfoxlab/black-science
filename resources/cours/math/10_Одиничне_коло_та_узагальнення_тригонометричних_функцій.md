# Одиничне коло та узагальнення тригонометричних функцій

## 2.1 Одиничне коло

### 2.1.1 Визначення

Одиничне коло - коло з центром у початку координат і радіусом `1`.

### 2.1.2 Побудова точок на колі

Кут відкладають від додатного напряму осі `Ox`. Додатний напрям - проти годинникової стрілки.

### 2.1.3 Координати точок

Точка на одиничному колі, що відповідає куту `x`, має координати:

```text
(cos x; sin x)
```

## 2.2 Тригонометричні функції будь-якого кута

### 2.2.1 Синус довільного кута

`sin x` - ордината точки одиничного кола.

### 2.2.2 Косинус довільного кута

`cos x` - абсциса точки одиничного кола.

### 2.2.3 Тангенс довільного кута

```text
tg x = sin x / cos x
```

Визначений, якщо `cos x != 0`.

### 2.2.4 Котангенс довільного кута

```text
ctg x = cos x / sin x
```

Визначений, якщо `sin x != 0`.

## 2.3 Знаки тригонометричних функцій за квадрантами

### 2.3.1 I квадрант

Усі функції додатні.

### 2.3.2 II квадрант

`sin` додатний, `cos`, `tg`, `ctg` від'ємні.

### 2.3.3 III квадрант

`tg` і `ctg` додатні, `sin` і `cos` від'ємні.

### 2.3.4 IV квадрант

`cos` додатний, `sin`, `tg`, `ctg` від'ємні.

## 2.4 Періодичність тригонометричних функцій

### 2.4.1 Період синуса

```text
sin(x + 2π) = sin x
```

### 2.4.2 Період косинуса

```text
cos(x + 2π) = cos x
```

### 2.4.3 Період тангенса та котангенса

```text
tg(x + π) = tg x
ctg(x + π) = ctg x
```

### 2.4.4 Знаходження найменшого позитивного періоду

Для `sin(kx)` і `cos(kx)`:

```text
T = 2π/|k|
```

Для `tg(kx)` і `ctg(kx)`:

```text
T = π/|k|
```
