# Формули додавання та їх наслідки

## 4.1 Формули синуса та косинуса суми й різниці

### 4.1.1 `sin(α ± β)`

```text
sin(α + β) = sin α cos β + cos α sin β
sin(α - β) = sin α cos β - cos α sin β
```

### 4.1.2 `cos(α ± β)`

```text
cos(α + β) = cos α cos β - sin α sin β
cos(α - β) = cos α cos β + sin α sin β
```

### 4.1.3 Застосування формул

Формули використовують для обчислення точних значень, спрощення виразів і доведення тотожностей.

## 4.2 Формули тангенса та котангенса суми й різниці

### 4.2.1 `tg(α ± β)`

```text
tg(α + β) = (tg α + tg β)/(1 - tg α tg β)
tg(α - β) = (tg α - tg β)/(1 + tg α tg β)
```

### 4.2.2 `ctg(α ± β)`

Котангенс можна отримати через `ctg x = 1/tg x` або спеціальні формули.

## 4.3 Формули подвійного кута

### 4.3.1 `sin(2α)`

```text
sin 2α = 2sin α cos α
```

### 4.3.2 `cos(2α)`

```text
cos 2α = cos^2 α - sin^2 α
```

### 4.3.3 `cos(2α) = 2cos²α - 1`

Отримують із `sin^2 α + cos^2 α = 1`.

### 4.3.4 `cos(2α) = 1 - 2sin²α`

Зручна, коли у виразі є синус.

### 4.3.5 `tg(2α)`

```text
tg 2α = 2tg α/(1 - tg^2 α)
```

## 4.4 Формули потрійного кута

### 4.4.1 `sin(3α)`

```text
sin 3α = 3sin α - 4sin^3 α
```

### 4.4.2 `cos(3α)`

```text
cos 3α = 4cos^3 α - 3cos α
```

### 4.4.3 `tg(3α)`

```text
tg 3α = (3t - t^3)/(1 - 3t^2), де t = tg α
```

## 4.5 Формули половинного кута

### 4.5.1 `sin(α/2)`

```text
sin^2(α/2) = (1 - cos α)/2
```

### 4.5.2 `cos(α/2)`

```text
cos^2(α/2) = (1 + cos α)/2
```

### 4.5.3 `tg(α/2)`

```text
tg(α/2) = sin α/(1 + cos α) = (1 - cos α)/sin α
```

### 4.5.4 `ctg(α/2)`

```text
ctg(α/2) = sin α/(1 - cos α) = (1 + cos α)/sin α
```

## 4.6 Формули суми й різниці синусів та косинусів

### 4.6.1 Сума синусів

```text
sin α + sin β = 2sin((α+β)/2)cos((α-β)/2)
```

### 4.6.2 Різниця синусів

```text
sin α - sin β = 2cos((α+β)/2)sin((α-β)/2)
```

### 4.6.3 Сума косинусів

```text
cos α + cos β = 2cos((α+β)/2)cos((α-β)/2)
```

### 4.6.4 Різниця косинусів

```text
cos α - cos β = -2sin((α+β)/2)sin((α-β)/2)
```

## 4.7 Формули добутку синусів та косинусів

### 4.7.1 `sin α sin β`

```text
sin α sin β = (cos(α-β) - cos(α+β))/2
```

### 4.7.2 `cos α cos β`

```text
cos α cos β = (cos(α-β) + cos(α+β))/2
```

### 4.7.3 `sin α cos β`

```text
sin α cos β = (sin(α+β) + sin(α-β))/2
```
