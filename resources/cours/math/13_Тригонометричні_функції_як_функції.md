# Тригонометричні функції як функції дійсного числа

## 5.1 Синус як функція

### 5.1.1 Область визначення

`sin x` визначений для всіх дійсних `x`.

### 5.1.2 Область значень

```text
-1 <= sin x <= 1
```

### 5.1.3 Графік

Графік синуса - синусоїда.

### 5.1.4 Властивості

Синус непарний: `sin(-x) = -sin x`. Період `2π`.

### 5.1.5 Монотонність

На відрізку `[-π/2; π/2]` синус зростає.

## 5.2 Косинус як функція

### 5.2.1 Область визначення

`cos x` визначений для всіх дійсних `x`.

### 5.2.2 Область значень

```text
-1 <= cos x <= 1
```

### 5.2.3 Графік

Графік косинуса - косинусоїда.

### 5.2.4 Властивості

Косинус парний: `cos(-x) = cos x`. Період `2π`.

### 5.2.5 Монотонність

На `[0; π]` косинус спадає.

## 5.3 Тангенс як функція

### 5.3.1 Область визначення

```text
x != π/2 + πn
```

### 5.3.2 Область значень

Усі дійсні числа.

### 5.3.3 Графік

Графік має вертикальні асимптоти `x = π/2 + πn`.

### 5.3.4 Властивості

Тангенс непарний, період `π`.

### 5.3.5 Монотонність

Тангенс зростає на кожному проміжку області визначення.

## 5.4 Котангенс як функція

### 5.4.1 Область визначення

```text
x != πn
```

### 5.4.2 Область значень

Усі дійсні числа.

### 5.4.3 Графік

Має вертикальні асимптоти `x = πn`.

### 5.4.4 Властивості

Котангенс непарний, період `π`.

### 5.4.5 Монотонність

Котангенс спадає на кожному проміжку області визначення.

## 5.5 Трансформації графіків тригонометричних функцій

### 5.5.1 `y = A sin(Bx + C) + D`

`A` задає амплітуду, `B` впливає на період, `C` - фазовий зсув, `D` - вертикальний зсув.

### 5.5.2 `y = A cos(Bx + C) + D`

Період:

```text
T = 2π/|B|
```

### 5.5.3 Амплітуда, період, фаза, зміщення

Амплітуда `|A|`, період `2π/|B|`, вертикальне зміщення `D`, горизонтальний зсув розв'язують із `Bx + C = 0`.
