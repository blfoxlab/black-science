# Логарифми

## 3.1 Визначення логарифма

Логарифм відповідає на питання: "До якого степеня треба піднести основу, щоб отримати число?"

### 3.1.1 Основні поняття

```text
log_a b = c  <=>  a^c = b
```

Умови існування:

- `a > 0`;
- `a != 1`;
- `b > 0`.

### 3.1.2 Знаходження логарифма

Щоб знайти логарифм, треба перетворити запис на показникову форму. Наприклад:

```text
log_2 8 = 3, бо 2^3 = 8
log_3 1/9 = -2, бо 3^(-2) = 1/9
```

### 3.1.3 Основна логарифмічна тотожність

```text
a^(log_a x) = x
log_a(a^x) = x
```

Це працює за умов `a > 0`, `a != 1`, `x > 0` для першої рівності.

## 3.2 Властивості логарифмів

Властивості логарифмів дозволяють перетворювати множення на додавання, ділення на віднімання, степінь на множник.

### 3.2.1 Логарифм добутку

```text
log_a(xy) = log_a x + log_a y
```

де `x > 0`, `y > 0`.

### 3.2.2 Логарифм частки

```text
log_a(x/y) = log_a x - log_a y
```

де `x > 0`, `y > 0`.

### 3.2.3 Логарифм степеня

```text
log_a(x^k) = k log_a x
```

На НМТ часто використовують у зворотному напрямі: `2log_a x = log_a(x^2)`.

### 3.2.4 Логарифм кореня

```text
log_a n√x = (1/n)log_a x
```

бо `n√x = x^(1/n)`.

### 3.2.5 Формула переходу до нової основи

```text
log_a x = log_b x / log_b a
```

Особливо корисно для обчислення через десяткові або натуральні логарифми.

## 3.3 Десятковий логарифм

### 3.3.1 Визначення та властивості

Десятковий логарифм має основу `10`:

```text
lg x = log_10 x
```

Наприклад, `lg 1000 = 3`, бо `10^3 = 1000`.

### 3.3.2 Характеристика та мантиса

У поглибленому курсі десятковий логарифм поділяють на цілу частину (характеристику) і дробову частину (мантису). Для НМТ зазвичай достатньо вміти обчислювати прості значення й застосовувати властивості.

## 3.4 Натуральний логарифм

### 3.4.1 Число e

Число `e ≈ 2.71828` - основа натурального логарифма. Воно виникає в задачах росту, похідних, інтегралах.

### 3.4.2 Властивості натурального логарифма

```text
ln x = log_e x
ln e = 1
ln 1 = 0
```

Натуральний логарифм має ті самі властивості, що й будь-який логарифм.
