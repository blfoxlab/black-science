# Похідна функції

## 2.1 Поняття похідної

### 2.1.1 Визначення похідної

```text
f'(x) = lim(Δx→0) (f(x+Δx)-f(x))/Δx
```

### 2.1.2 Геометричний смисл

Похідна в точці - кутовий коефіцієнт дотичної до графіка.

### 2.1.3 Механічний смисл

Похідна координати за часом - миттєва швидкість.

### 2.1.4 Диференційованість та неперервність

Якщо функція диференційована, то вона неперервна. Але неперервність не завжди гарантує похідну.

## 2.2 Правила диференціювання

### 2.2.1 Похідна сталої

```text
(C)' = 0
```

### 2.2.2 Похідна суми

```text
(f + g)' = f' + g'
```

### 2.2.3 Похідна добутку

```text
(fg)' = f'g + fg'
```

### 2.2.4 Похідна частки

```text
(f/g)' = (f'g - fg')/g^2
```

### 2.2.5 Похідна складної функції

```text
(f(g(x)))' = f'(g(x))*g'(x)
```

### 2.2.6 Похідна оберненої функції

Якщо існує обернена функція, то її похідну можна знаходити через похідну початкової.

### 2.2.7 Логарифмічне диференціювання

Зручно для виразів зі степенями, добутками й частками: спочатку беруть логарифм, потім диференціюють.

## 2.3 Таблиця похідних основних функцій

```text
(x^n)' = nx^(n-1)
(sin x)' = cos x
(cos x)' = -sin x
(tg x)' = 1/cos^2 x
(ctg x)' = -1/sin^2 x
(e^x)' = e^x
(a^x)' = a^x ln a
(ln x)' = 1/x
(log_a x)' = 1/(x ln a)
(arcsin x)' = 1/√(1-x^2)
(arccos x)' = -1/√(1-x^2)
(arctg x)' = 1/(1+x^2)
(arcctg x)' = -1/(1+x^2)
```

## 2.4 Диференціал функції

### 2.4.1 Визначення диференціала

```text
dy = f'(x)dx
```

### 2.4.2 Геометричний смисл

Диференціал наближено показує приріст функції вздовж дотичної.

### 2.4.3 Властивості диференціала

Підпорядковується правилам похідної.

### 2.4.4 Наближені обчислення

```text
f(x + Δx) ≈ f(x) + f'(x)Δx
```

## 2.5 Похідні вищих порядків

### 2.5.1 Друга похідна

Похідна від похідної: `f''(x)`.

### 2.5.2 Третя та вищі похідні

Обчислюються послідовним диференціюванням.

### 2.5.3 Формула Лейбніца

Поглиблена формула для похідної добутку високого порядку.

### 2.5.4 Інтерпретація другої похідної

Друга похідна описує опуклість графіка й прискорення в механіці.
