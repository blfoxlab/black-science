# Ймовірність

## 2.1 Основні поняття

### 2.1.1 Експеримент та випадкова подія

Випадкова подія може відбутися або не відбутися внаслідок експерименту.

### 2.1.2 Елементарна подія

Найпростіший результат експерименту.

### 2.1.3 Простір елементарних подій

Множина всіх можливих елементарних подій, позначається `Ω`.

### 2.1.4 Вірогідна та неможлива подія

Вірогідна має ймовірність `1`, неможлива - `0`.

## 2.2 Операції над подіями

### 2.2.1 Об'єднання

`A ∪ B` - відбулася хоча б одна із подій.

### 2.2.2 Перетин

`A ∩ B` - відбулися обидві події.

### 2.2.3 Протилежна подія

`не A`: подія `A` не відбулася.

### 2.2.4 Несумісні та незалежні події

Несумісні не можуть відбутися одночасно. Незалежні не впливають одна на одну.

## 2.3 Класичне визначення

### 2.3.1 Формула

```text
P(A) = |A|/|Ω|
```

### 2.3.2 Властивості

`0 <= P(A) <= 1`, `P(Ω)=1`, `P(не A)=1-P(A)`.

### 2.3.3 Вибір результатів

Важливо правильно порахувати всі можливі та сприятливі випадки.

## 2.4 Геометрична ймовірність

### 2.4.1 Вибір точки

Ймовірність дорівнює відношенню довжин, площ або об'ємів.

### 2.4.2 Задачі

```text
P = площа сприятливої області / площа всієї області
```

### 2.4.3 Площа та об'єм

У просторових задачах аналогічно використовують об'єми.

## 2.5 Теорема додавання

### 2.5.1 Несумісні події

```text
P(A ∪ B) = P(A) + P(B)
```

### 2.5.2 Сумісні події

```text
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
```

### 2.5.3 Багато подій

Використовують принцип включення-виключення.

## 2.6 Умовна ймовірність

### 2.6.1 Визначення

Ймовірність `A` за умови, що `B` уже відбулася.

### 2.6.2 Формула

```text
P(A|B) = P(A ∩ B)/P(B)
```

### 2.6.3 Обчислення

Звужуємо простір можливих результатів до події `B`.

## 2.7 Теорема множення

### 2.7.1 Залежні події

```text
P(A ∩ B) = P(A)P(B|A)
```

### 2.7.2 Незалежні події

```text
P(A ∩ B) = P(A)P(B)
```

### 2.7.3 Кілька подій

Для незалежних подій множимо ймовірності всіх подій.

## 2.8 Формула повної ймовірності

### 2.8.1 Розбиття простору

Події `B_i` мають бути несумісні й разом покривати весь простір.

### 2.8.2 Формула

```text
P(A) = Σ P(B_i)P(A|B_i)
```

### 2.8.3 Застосування

Коли подія може статися через кілька сценаріїв.

## 2.9 Формула Баєса

### 2.9.1 Запис правила

```text
P(B_i|A) = P(A|B_i)P(B_i)/P(A)
```

### 2.9.2 Застосування

Для переоцінки ймовірності причини після того, як подія відбулася.

### 2.9.3 Обернена задача

Відомий результат, треба знайти ймовірність причини.

## 2.10 Схема Бернуллі

### 2.10.1 Незалежні випробування

Є `n` однакових незалежних випробувань, у кожному успіх із ймовірністю `p`.

### 2.10.2 Формула

```text
P_n(k) = C_n^k p^k(1-p)^(n-k)
```

### 2.10.3 Найбільш імовірна кількість успіхів

Знаходиться біля числа `(n+1)p`.

### 2.10.4 Локальна теорема Лапласа

Поглиблене наближення біноміального розподілу.

### 2.10.5 Інтегральна теорема Лапласа

Використовується для ймовірності потрапляння кількості успіхів у проміжок.

### 2.10.6 Теорема Пуассона

Застосовується для рідкісних подій при великому `n` і малому `p`.
