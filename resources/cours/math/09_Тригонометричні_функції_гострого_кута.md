# Тригонометричні функції гострого кута

## 1.1 Визначення синуса, косинуса, тангенса, котангенса

### 1.1.1 У прямокутному трикутнику

Тригонометричні функції гострого кута визначають через відношення сторін прямокутного трикутника.

### 1.1.2 Відношення сторін

```text
sin α = протилежний катет / гіпотенуза
cos α = прилеглий катет / гіпотенуза
tg α = протилежний катет / прилеглий катет
ctg α = прилеглий катет / протилежний катет
```

### 1.1.3 Залежність від кута

Для того самого кута значення тригонометричних функцій однакові в усіх подібних прямокутних трикутниках.

## 1.2 Основні тригонометричні тотожності

### 1.2.1 `sin²α + cos²α = 1`

Головна тотожність випливає з теореми Піфагора.

### 1.2.2 `tg(α) = sin(α)/cos(α)`

Визначено, якщо `cos α != 0`.

### 1.2.3 `ctg(α) = cos(α)/sin(α)`

Визначено, якщо `sin α != 0`.

### 1.2.4 `tg(α)·ctg(α) = 1`

Працює, коли обидві функції визначені.

### 1.2.5 `1 + tg²(α) = sec²(α)`

У шкільному записі:

```text
1 + tg^2 α = 1/cos^2 α
```

### 1.2.6 `1 + ctg²(α) = cosec²(α)`

У шкільному записі:

```text
1 + ctg^2 α = 1/sin^2 α
```

## 1.3 Значення тригонометричних функцій особливих кутів

### 1.3.1 `30°`, `45°`, `60°`

Ці кути треба знати напам'ять, бо вони часто трапляються в геометрії та тригонометрії.

### 1.3.2 Таблиця значень

| Кут | `30°` | `45°` | `60°` |
|---|---:|---:|---:|
| `sin` | `1/2` | `√2/2` | `√3/2` |
| `cos` | `√3/2` | `√2/2` | `1/2` |
| `tg` | `√3/3` | `1` | `√3` |
| `ctg` | `√3` | `1` | `√3/3` |

### 1.3.3 Градусна та радіанна міра кутів

```text
180° = π
90° = π/2
60° = π/3
45° = π/4
30° = π/6
```
