# Диференціальні рівняння

## 5.1 Основні поняття

### 5.1.1 Визначення диференціального рівняння

Диференціальне рівняння містить невідому функцію та її похідні.

### 5.1.2 Порядок

Порядок визначається найвищою похідною.

### 5.1.3 Розв'язок рівняння

Розв'язок - функція, яка перетворює рівняння на правильну рівність.

### 5.1.4 Загальний та частковий розв'язок

Загальний містить сталі. Частковий отримують після підстановки початкових умов.

## 5.2 Рівняння першого порядку

### 5.2.1 З розділюваними змінними

Мають вигляд, де можна зібрати `y` з `dy`, а `x` з `dx`, потім проінтегрувати.

### 5.2.2 Однорідні рівняння

Розв'язують заміною `y = ux`.

### 5.2.3 Лінійні рівняння першого порядку

Мають вигляд `y' + p(x)y = q(x)`.

### 5.2.4 Рівняння Бернуллі

Поглиблена тема, зводиться до лінійного рівняння спеціальною заміною.

## 5.3 Рівняння вищих порядків

### 5.3.1 Зниження порядку

Якщо рівняння має спеціальний вигляд, порядок можна зменшити заміною.

### 5.3.2 Лінійні однорідні рівняння

Для сталих коефіцієнтів розв'язують через характеристичне рівняння.

### 5.3.3 Лінійні неоднорідні рівняння

Розв'язок = загальний розв'язок однорідного + частковий розв'язок неоднорідного.

## 5.4 Рівняння другого порядку

### 5.4.1 Характеристичне рівняння

Для `ay'' + by' + cy = 0`:

```text
ar^2 + br + c = 0
```

### 5.4.2 Розв'язок однорідного рівняння

Залежить від коренів характеристичного рівняння.

### 5.4.3 Частковий розв'язок неоднорідного

Підбирають або знаходять спеціальними методами. Для НМТ це довідкова тема.
